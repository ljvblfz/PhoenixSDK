Easy Plugin

 

 

Sample Name: EasyPlugin

 

Sample Category: plugin

 

Description: The EasyPlugin contains code for performing various basic tasks using Phoenix’s plugin mechanism.

 

Source Language: C#

 

Sample Location: Samples\EasyPlugin\csharp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\EasyPlugin\csharp

  devenv EasyPlugin.csproj /build debug

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\Identity.cpp -d2 plugin:bin\debug\EasyPlugin.dll

 

Detailed Description: This sample shows how to display function names, count instructions, count operands, dump opcodes, and dump the entire IR code of a function unit.
