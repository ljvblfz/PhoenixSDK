Hello Sample

 

 

Sample Name: Hello

 

Sample Category: Application

 

Description: hello creates an object file from scratch for a simple C++ program.

 

Source Language: C#, C++/CLI

 

Sample Location: Samples\hello\csharp,  Samples\hello\cpp

 

Usage: From the command line:

 

hello

 

Detailed Description: The Hello sample creates an object file for a simple C++ program from scratch. The resulting object file can be linked to build a fully functional and debuggable executable.  Hello illustrates creation of symbols, types, debug information, IR, and initialized global variables.  Hello can also host plugins and is responsive to debug controls. The source code for Hello also includes a series of interesting exercises.
