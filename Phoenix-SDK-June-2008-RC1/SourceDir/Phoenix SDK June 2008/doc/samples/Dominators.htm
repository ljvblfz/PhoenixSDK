Dominators

 

 

Sample Name: Dominators

 

Sample Category: plugin

 

Description: The plugin solves a dataflow analysis problem to build the dominance relation in the control flow graph of each function unit.

 

Source Language: C++

 

Sample Location: Samples\Dominators\cpp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\Dominators\cpp

  devenv Dominators.vcproj /build

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\SimpleNestedLoops.cpp -d2 plugin:debug\Dominators.dll

 

Detailed Description:

 

This sample uses the DataFlow API to resolve the following defining equations of the dominance relation by fixed-point iteration.

 

Dom(B) = {B} È (ÇP in Pred(B) Dom(P)), B ≠ Start

Dom(Start) = { Start }

 

Sets are represented by sparse vectors.
