Natural Loop Bodies

 

 

Sample Name: NaturalLoopBodies

 

Sample Category: plugin

 

Description: This plugin finds the headers of natural loops and computes the set of nodes reachable through inverse edges by starting at back edges.

 

Source Language: C++

 

Sample Location: Samples\NaturalLoopBodies\cpp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\NaturalLoopBodies\cpp

  devenv NaturalLoopBodies.vcproj /build

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\SimpleNestedLoops.cpp -d2 plugin:debug\NaturalLoopBodies.dll

 

Detailed Description:

 

This sample finds headers of natural loops in a way entirely analogous to sample NaturalLoops.

 

The body (or, in the case of nested loops, the bodies) dominated by a loop header is (are) found by (a) finding the header’s predecessor(s) through back-edges and (b) computing the set of nodes reachable backwards from each such predecessor.

 

This plugin illustrates how to use a sparse bitmap as a worklist to keep track of the set of nodes whose predecessors have to be visited.
