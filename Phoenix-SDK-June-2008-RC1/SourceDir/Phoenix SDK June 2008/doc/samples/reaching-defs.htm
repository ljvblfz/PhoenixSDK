Reaching Defs Sample

 

 

Sample Name: Reaching Defs

 

Sample Category: Code Analysis Plug-In

 

Description: It computes the reaching definitions for all the basic blocks in a function.

 

Source Language: C++/CLI

 

Sample Location: Samples\reaching-defs\cpp

 

Usage: Uses the standard Phoenix plugin model for c2.  Example:

 

cl -d2plugin:<path to reaching-defs.dll>  ...

 

Detailed Description: The Reaching Defs plugin sample demonstrates dynamic extensibility through plugins, controls, and phases. It is an example of a program which analyzes the IR.  A phase (Reaching Definitions Phase) is added after the MIR Lower phase. Execute() creates a control flow graph, identifies the definitions to be tracked and then computes the in and out set for each basic block in the function. 

 

The sample will produce an output without any switches and will add an extra info with -d2verbosetrace:ReachingDefs switch for debug and release.
