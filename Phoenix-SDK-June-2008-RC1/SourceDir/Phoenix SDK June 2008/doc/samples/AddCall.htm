AddCall Sample

 

 

Sample Name: AddCall

 

Sample Category: Instrumentation/Code Transformation

 

Description : The AddCall sample opens a native (unmanaged) x86 binary and writes out a new binary that contains a call, inserted at the beginning of a specified local method, to an external method in a DLL.

 

Source Language: C++/CLI

 

Sample Location: Samples\addcall\cpp

 

Usage::From the command line:

 

addcall /in <input binary>

/importdll <DLL containing method to call>

/importmethod <name of method to call>

/localmethod <name of local method to add call to>

/out <output binary>

/pdbout <output PDB>

 

 

Detailed Description: The AddCall sample inserts a new import to the external method, specified by /importmethod in the DLL specified by /importdll, and then a call to that import is inserted to the beginning of the method specified by /localmethod.  The external method should have a return type of void and take no parameters.

 

As an example, one could use the ProcTraceRuntime from the ProcTrace sample. So, for this example, assume that hello.exe is a simple console application. To add a call from the beginning of main() in hello.exe to the Shout() method of ProcTraceRuntime.dll:

 

addcall /out newtest.exe /pdbout newtest.pdb

/importdll proctraceruntime.dll /importmethod _Shout@0

/localmethod main /in hello.exe
