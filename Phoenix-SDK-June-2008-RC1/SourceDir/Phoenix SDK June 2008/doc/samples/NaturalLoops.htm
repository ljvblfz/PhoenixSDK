Natural Loops

 

 

Sample Name: NaturalLoops

 

Sample Category: plugin

 

Description: The plugin searches for back edges in the flow graph so as to identify the headers of natural loops.

 

Source Language: C++

 

Sample Location: Samples\NaturalLoops\cpp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\NaturalLoops\cpp

  devenv NaturalLoops.vcproj /build

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\SimpleNestedLoops.cpp -d2 plugin:debug\NaturalLoops.dll

 

Detailed Description:

 

This sample uses Phoenix functions to build a flow graph, build a post order node list for this flow graph, and build a dominance relation.

 

It uses these ingredients to perform an efficient search for back edges, where a back edge is defined as an edge whose target dominates the source. Targets of back-edges are headers of natural loops.
