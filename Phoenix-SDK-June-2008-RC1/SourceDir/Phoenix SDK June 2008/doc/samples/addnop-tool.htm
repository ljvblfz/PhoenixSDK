AddNop Tool Sample

 

Sample Name: AddNop-tool

 

Sample Category: Instrumentation/Code Transformation

 

Description: The AddNop tool takes as input a Phoenix compatible x86 binary and produces as output an x86 binary in which an x86 nop instruction has been added before each x86 instruction in the source file. It process x86, MSIL or mixed binaries.

 

Source Language: C#, C++/CLI

 

Sample Location: Samples\addnop-tool\csharp, Samples\addnop-tool\cpp

 

Usage: From the command line:

 

AddNop-tool [/ShowResults] <src-file> <dest-file>

 

      /ShowResults - If enabled, it will dump IR for every funcUnit before and after instrumentation.

                     This control is an optional argument.

 

Detailed Description: The AddNop-tool sample is a simple example of a tool which transforms a binary.  In the case of AddNop-tool, the transformation is a trivial one in which an x86 Nop instruction is added before each existing x86 instruction in the source code.  The main “action” of AddNop-tool can be found in the DoAddNop() method where a simple loop iterates over all the instructions in the input binary and produces a new binary with the Nop inserted.
