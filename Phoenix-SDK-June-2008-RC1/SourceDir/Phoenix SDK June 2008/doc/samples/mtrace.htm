MTrace Sample

 

 

Sample Name: MTrace

 

Sample Category: Application

 

Description: Inject tracing code into a managed application.

 

Source Language: C#, C++/CLI

 

Sample Location: Samples\mtrace\csharp,  Samples\mtrace\cpp

 

Usage: From the command line:

 

mtrace <assembly-file>

 

Detailed Description: MTrace demonstrates how to add tracing code into a managed application. Each method in the input assembly is modified to call the Systems.Diagnostics.Trace.WriteLine on entry and exit to each method. Mtrace creates a string containing the method name and source location as the WriteLine argument.
