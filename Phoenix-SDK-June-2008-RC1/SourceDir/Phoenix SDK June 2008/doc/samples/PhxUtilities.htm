Phoenix Utilities

 

 

Sample Name: PhxUtilities

 

Sample Category: Class Library

 

Description: This class library provides utilities for namespace name resolution and building dependency graphs.

 

Source Language: C#

 

Sample Location: Samples\PhxUtilities\Csharp

 

Usage: As a class library, this is not used directly.  Example:

 

cd %PhoenixSdkUserRoot%\samples\PhxUtilities\csharp

devenv PhxUtilities.csproj /build

 

Detailed Description:

 

This class library contains three separate utilities. The first is a static class which is useful for namespace name resolution. This provides functionality which will build the fully qualified name of a function and determine the namespace of a function. The second is a new type of graph. This graph is an overlay graph which is intended to be used on top of other graphs. Overlay graphs have dictionaries and lists for storing relations to other graphs. The last utility is an example use of the overlay graph. This utility builds dependency graphs which represent the relationships between functions, classes, namespaces, and files.
