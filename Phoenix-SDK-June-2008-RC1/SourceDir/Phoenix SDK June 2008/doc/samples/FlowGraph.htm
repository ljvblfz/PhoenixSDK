Flow Graph

 

 

Sample Name: FlowGraph

 

Sample Category: plugin

 

Description: This plugin shows how to build a function unit’s flow graph using IR information.

 

Source Language: C#

 

Sample Location: Samples\FlowGraph\csharp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\FlowGraph\csharp

  devenv FlowGraph.csproj /build debug

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\Identity.cpp -d2 plugin:bin\debug\FlowGraph.dll

 

Detailed Description:

 

This plugin illustrates how to use edge information embedded in Phx.IR.Instruction objects to identify basic blocks and edges.
