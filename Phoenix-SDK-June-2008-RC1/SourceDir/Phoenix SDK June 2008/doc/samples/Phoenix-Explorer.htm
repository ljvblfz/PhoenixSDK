PE-Explorer Sample

 

 

Sample Name: PE-Explorer

 

Sample Category: Application

 

Description: A GUI application for exploring PE files.

 

Source Language: C#, C++/CLI

 

Sample Location: Samples\PE-Explorer\csharp

                                                                                                                          

Usage: From the command line:

 

pe-explorer

 

Detailed Description: PE-Explorer is a GUI application for browsing PE files. It illustrates how to read PE images and browse them in a hierarchical fashion, and then examine the IR for various methods within the image.
