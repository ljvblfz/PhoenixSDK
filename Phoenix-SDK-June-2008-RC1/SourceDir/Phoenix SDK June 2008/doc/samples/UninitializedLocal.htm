UninitializedLocal Sample

 

 

Sample Name: UninitializedLocal

 

Sample Category: c2 plug-in

 

Description: Checks for uses of uninitialized local variables.

 

Source Language: C++/CLI

 

Sample Location: Samples\UninitializedLocal\cpp

 

Usage: Uses the standard Phoenix plugin model for c2.  Example:

 

cl -d2plugin:<path to UninitializedLocal.dll> [-d2WarnMayUninit] ...

 

Detailed Description: UninitializedLocal is a plugin which replaces Phoenix’s phase for detecting uses of uninitialized local variables with its own version. It uses Single Static Assignment (SSA) information to find the uninitialized uses, and adds a command-line switch (-WarnMayUninit) to turn on warnings for variables which are initialized through some code paths and not others.
