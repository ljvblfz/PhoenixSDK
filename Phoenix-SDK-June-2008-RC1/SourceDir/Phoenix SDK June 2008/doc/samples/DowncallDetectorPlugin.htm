Namespace Downcall Detector

 

 

Sample Name: DowncallDetectorPlugin

 

Sample Category: plug-in

 

Description: Detects downward calls in the namespace hierarchy and signals warnings.

 

Source Language: C#

 

Sample Location: Samples\DowncallDetectorPlugin\Csharp

 

Usage: Uses the standard Phoenix plugin model for pereader.  Example:

 

pereader /callgraph /plugin DowncallDetectorPlugin.dll [My Binary].[exe|dll]

 

Detailed Description:

 

This plugin is intended to detect function calls down the namespace hierarchy and signal them with warnings. The plug-in warns the user when a software-engineering principle is violated. Generally, definitions within a parent namespace should not depend on definitions within a child namespace. Such a dependency violates some of the usefulness of the abstraction produced by a namespace hierarchy. This plug-in looks over all the edges in the call graph and determines which of those edges are in violation. It uses the PhxUtilities class library for namespace name resolution. This plugin for pereader may also work with C2 and could be used to generate more warnings within visual studio.
