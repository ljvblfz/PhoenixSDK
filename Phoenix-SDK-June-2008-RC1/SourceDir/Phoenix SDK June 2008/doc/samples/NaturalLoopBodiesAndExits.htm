Natural Loop Bodies and Exits

 

 

Sample Name: NaturalLoopBodiesAndExits

 

Sample Category: plugin

 

Description: This plugin finds the headers of natural loops and computes the set of nodes reachable through inverse edges by starting at back edges.

 

Source Language: C++

 

Sample Location: Samples\NaturalLoopBodiesAndExits\cpp

 

Usage: Uses the standard Phoenix plugin model for C2.  Example:

 

  cd %PhoenixSdkUserRoot%\samples\NaturalLoopBodiesAndExits\cpp

  devenv NaturalLoopBodiesAndExits.vcproj /build debug

  cl -nologo -c %PhoenixSdkUserRoot%Applications\tests\src\cpp\SimpleNestedLoops.cpp -d2 plugin:debug\NaturalLoopBodiesAndExits.dll

 

Detailed Description:

 

This sample finds the sets of blocks of each natural loop in a way entirely analogous to sample NaturalLoopBodies.

 

Loop bodies with successor edges to blocks out of the loop are identified and their last instructions recorded as exits.
