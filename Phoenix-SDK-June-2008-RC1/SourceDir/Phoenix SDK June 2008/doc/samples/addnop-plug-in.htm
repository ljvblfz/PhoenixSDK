AddNop-plug-in Sample

 

 

Sample Name: AddNop-plug-in

 

Sample Category: Instrumentation/Code Transformation Plug-In

 

Description : The AddNop plugin inserts Nop instructions before every real instruction in the IR.

 

Source Language: C#, C++/CLI

 

Sample Location: Samples\addnop-plug-in\csharp,  Samples\addnop-plug-in\cpp

 

Usage: Uses the standard Phoenix plugin model for c2.  Example:

 

cl -d2plugin:<path to addnop-plug-in.dll>  ...

 

Detailed Description: The AddNop plugin sample is a simple example of a program which transforms the IR. The sample is packaged as a Phoenix plugin that can be used with any phase-based Phoenix executable. A phase (AddNOPPhase) is added after the Lower phase. AddNOPPhase walks the IR and inserts Nop instructions before each real instruction in the IR stream. The sample demonstrates dynamic extensibility through plugins, controls, and phases.
